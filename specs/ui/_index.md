Content:

- `views/` - Specifications for the views.  A view is a page of the application that the user can navigate to, such as the landing page, about page, etc.

- `components/` - Specifications for components of the application, such as the statusbar, user profile, etc.  These are reusable across views.

- `style` - Contains the style guide for the application, including color scheme, typography, and other design elements.
